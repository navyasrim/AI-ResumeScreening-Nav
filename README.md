# AI-ResumeScreening-Nav
An AI-powered HR tool that accepts job descriptions and batches of resumes (PDF), scores each candidate against the JD using gemini-2.5-flash-lite, ranks them with structured reasoning, and generates personalized interview questions tailored to each candidate's background and skill gaps.
